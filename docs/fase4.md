# Fase 4 — Execução e Análise dos Resultados

## Introdução

Esta fase documenta a **execução das medições de qualidade** do software MEPA, conforme detalhado no **Planejamento da Avaliação (Fase 3)**. 

O objetivo é verificar e evidenciar a qualidade do sistema frente às características priorizadas: 

- **Adequação Funcional (AF)**
- **Eficiência de Desempenho (PE)**
- **Manutenibilidade (MT)**

Os resultados são apresentados na ordem das prioridades definidas nas Fases 1 e 2.
